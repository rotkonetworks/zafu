05
rules · clear-space · do · don't
restraint over ornament
clear-space
minimum padding around any lockup = height of the glyph. at 16px favicon, no padding. never place on non-black backgrounds without a #050505 card behind it.
do
use on #000.
rebind the active-dot colour to the current chain.
keep the slash / in fg-dim, never in gold.
don't
never tilt, skew, or 3d the glyph.
never recolour zafu itself — only the wallet half.
never pair with emoji or illustrative icons.
