rules
restraint over range
keep
one accent per view. rebind only through [data-network].
borders, never shadows, express elevation.
tabular-nums on every figure, hash and timestamp.
avoid
gradients · aurora backgrounds · rainbow badges.
coloured body text. grey shades only.
soft pastels · anything that reads "fintech".
instead of
a red error card → a single #ff5555 word.
a success toast → a #55ff55 dot next to the label.
multi-stop gradients → one colour on black.
